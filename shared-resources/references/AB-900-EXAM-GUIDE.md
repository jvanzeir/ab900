# AB-900: Microsoft 365 Copilot and Agent Administration Fundamentals - Complete Exam Guide

## Exam Overview

**Official Title:** Microsoft 365 Certified: Copilot and Agent Administration Fundamentals (beta)

**Exam Code:** AB-900

**Status:** Beta (as of 2025)

**Duration:** 45 minutes

**Passing Score:** 700 (out of 1000)

**Format:** Proctored exam with potential interactive components

**Language:** English

**Cost:** Standard Microsoft certification exam pricing (check Microsoft's website for current pricing)

## What This Certification Validates

This certification demonstrates your ability to:
- **Support, secure, and protect** an AI-enabled Microsoft 365 environment
- Work with Microsoft 365 core services including Exchange Online, SharePoint, Teams, Entra, and Purview
- Implement core security features including authentication and conditional access
- Manage and administer Microsoft 365 Copilot and agents
- Apply modern IT management practices

## Exam Skills Measured

### Domain 1: Core Microsoft 365 Features and Objects (30-35%)

#### License Management
- Understand how license assignment impacts feature access
- Assign and manage user licenses
- Monitor license usage and availability
- Understand Copilot licensing requirements (monthly and pay-as-you-go)

#### Microsoft 365 Admin Center
- Navigate and configure settings through the admin center
- Manage user accounts and groups
- Monitor service health
- Configure organizational settings

#### Exchange Online Administration
- Manage mailboxes (user, shared, resource)
- Create and manage distribution lists and groups
- Configure mail flow and transport rules
- Manage calendar permissions

#### SharePoint Administration
- Create and manage SharePoint sites
- Configure site permissions and sharing settings
- Manage site collections
- Implement SharePoint Advanced Management features
- Remediate oversharing issues

#### Microsoft Teams Administration
- Configure Teams policies and settings
- Manage team creation and membership
- Control app permissions
- Configure meeting policies

#### Zero Trust Security
- Understand Zero Trust principles
- Implement "never trust, always verify" methodology
- Apply least privilege access
- Use explicit verification

#### Microsoft Entra (Azure AD) Features
- Manage user identities and authentication
- Configure conditional access policies
- Implement multi-factor authentication (MFA)
- Set up single sign-on (SSO)
- Manage Privileged Identity Management (PIM) roles
- Understand eligible vs. active role assignments
- Configure just-in-time access

### Domain 2: Data Protection and Governance (35-40%)

#### Microsoft Purview Capabilities
- **Information Protection**
  - Create and apply sensitivity labels
  - Configure label policies
  - Implement automatic labeling
  - Manage encryption and access controls

- **Data Loss Prevention (DLP)**
  - Create DLP policies
  - Configure policy rules and conditions
  - Set up alerts and notifications
  - Monitor DLP reports
  - Respond to DLP alerts

- **Insider Risk Management**
  - Understand insider risk scenarios
  - Configure insider risk policies
  - Investigate insider risk alerts
  - Manage risk cases

- **Communication Compliance**
  - Set up communication compliance policies
  - Monitor organizational communications
  - Review and resolve compliance violations
  - Manage policy violations

- **Data Security Posture Management**
  - Assess data security posture
  - Identify security gaps
  - Implement security recommendations
  - Monitor security improvements

#### Copilot Data Access and Permissions
- Understand how Copilot accesses organizational data
- Configure Copilot data grounding settings
- Manage permissions for Copilot interactions
- Control web search in Copilot
- Implement data boundaries

#### Compliance Manager
- Navigate Compliance Manager interface
- Identify compliance risks
- Understand compliance score
- Use improvement actions
- Create and manage assessments
- Monitor compliance posture

#### Sensitive Data Discovery
- Use Content Explorer
- Utilize Activity Explorer
- Search for sensitive information
- Classify data automatically
- Create custom sensitive information types

#### SharePoint Oversharing Remediation
- Identify overshared content
- Use data access governance reports
- Implement restricted access controls
- Configure block download policies
- Conduct site access reviews

### Domain 3: Copilot and Agent Administration (25-30%)

#### Licensing Models
- **Monthly subscription** vs. **pay-as-you-go** billing
- Understand license prerequisites (M365 E3/E5, Business Standard/Premium)
- Assign Copilot licenses to users
- Manage billing policies
- Monitor license consumption

#### Feature Management
- Enable/disable Copilot features
- Configure Copilot settings in admin center
- Manage web search capabilities
- Control plugin access
- Set data sharing preferences

#### Agent Types and Use Cases
- **Researcher agents** - Information gathering and research
- **Analyst agents** - Data analysis and insights
- **Custom agents** - Organization-specific scenarios
- **Declarative agents** - Simple instruction-based agents

#### License Assignment Procedures
- Assign licenses individually
- Assign licenses to groups
- Use PowerShell for bulk assignment
- Monitor license activation (24-hour delay)
- Verify license status

#### Billing Policy Management
- Configure pay-as-you-go billing
- Set spending limits
- Monitor usage and costs
- Generate billing reports
- Manage budget alerts

#### Usage Monitoring and Adoption Metrics
- Access Copilot Dashboard in Viva Insights
- Track readiness metrics (licenses, activation, active users)
- Monitor adoption across apps (Teams, Outlook, Word, Excel, PowerPoint)
- Measure impact (Copilot assisted hours)
- Collect sentiment data
- Generate adoption reports
- Identify usage patterns

#### Prompt Management
- Understand prompt engineering best practices
- Create effective prompts for users
- Manage prompt libraries
- Share prompt examples
- Monitor prompt effectiveness

#### Agent Creation and Configuration
- Create declarative agents using Copilot Studio
- Configure agent instructions and persona
- Add knowledge sources to agents
- Configure agent actions and plugins
- Set conversation starters
- Test agents in development environment

#### Agent Approval Processes
- Configure agent publication policies
- Set up approval workflows
- Review agent submissions
- Manage agent permissions
- Control agent distribution
- Monitor agent compliance

## Exam Tips and Strategies

### Before the Exam
1. **Hands-On Practice is Essential** - This exam tests practical knowledge
2. **Use the Study Guide** - Focus on the official objectives
3. **Practice in a Real Tenant** - Get familiar with admin centers
4. **Understand Concepts, Not Just Facts** - Know the "why" behind configurations
5. **Review All Three Domains** - Don't skip any section

### During the Exam
1. **Read Questions Carefully** - Watch for words like "EXCEPT," "NOT," or "LEAST"
2. **Manage Your Time** - 45 minutes goes quickly
3. **Eliminate Wrong Answers** - Narrow down choices
4. **Mark Questions for Review** - Come back if unsure
5. **Don't Second-Guess** - First instinct is usually correct
6. **Look for Specific Details** - Questions may contain scenario details
7. **Stay Calm** - Take deep breaths if you feel stressed

### Question Types to Expect
- **Multiple choice** - Single correct answer
- **Multiple response** - Multiple correct answers (may not tell you how many)
- **Interactive scenarios** - Hands-on simulations
- **Drag and drop** - Ordering or matching
- **Case studies** - Multi-question scenarios

### Key Areas to Master
- License assignment and management
- DLP policy creation
- Sensitivity label configuration
- Copilot deployment steps
- Agent creation basics
- Security and compliance features
- Administrative centers navigation
- Troubleshooting common issues

## Common Pitfalls to Avoid

1. **Confusing Copilot Licensing** - Understand prerequisites and add-on models
2. **Overlooking Permissions** - Know which admin role is needed for each task
3. **Misunderstanding DLP vs. Sensitivity Labels** - Know when to use each
4. **Forgetting 24-Hour License Delay** - Copilot may not appear immediately
5. **Mixing Up Agent Types** - Understand differences between Researcher, Analyst, Custom
6. **Ignoring Prerequisites** - Know what's required before Copilot deployment
7. **Not Understanding Data Grounding** - How Copilot accesses organizational data
8. **Overlooking Compliance Manager** - Understand its role in risk identification
